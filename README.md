Stone Paper Scissor Game:
A simple and interactive Stone–Paper–Scissor game built using HTML, CSS, and JavaScript.
The user plays against the computer, and scores update live after every round.

Features:
✔️ Interactive UI
✔️ Click-based user choices (Stone / Paper / Scissor)
✔️ Random computer choice generation
✔️ Live scoreboard for both player and computer
✔️ Result message displayed instantly
✔️ Responsive layout

How the Game Works
1.The user selects Stone, Paper, or Scissor.
2.The computer randomly generates one choice.
3.Based on classic rules:
   Stone beats Scissor
   Paper beats Stone
   Scissor beats Paper
4.The winner is displayed, and scores update.
